# Mason Doney — Portfolio

A polished, single-page portfolio highlighting Mason Doney's web development background, Python automation strengths, and academic excellence at Lake Belton High School.

## Preview
Open `index.html` in your browser to explore the live experience. All featured content, skill badges, proof-point highlights, and timelines are generated automatically from structured data defined in `assets/js/app.js`.

## Structure
- `index.html` — Layout and semantic structure for the portfolio page with navigation and hero metrics.
- `assets/css/styles.css` — Visual design system, responsive grid, interactive styling, and sticky navigation.
- `assets/js/app.js` — Data-driven rendering for skills, projects, automation highlights, academic badges, proof points, and animated metrics.

## Notes
- Metrics animate on load to showcase project count, automation hours saved, and GPA performance.
- Content sections emphasize responsive web builds, automation workflows, and high grades from Lake Belton High School.
